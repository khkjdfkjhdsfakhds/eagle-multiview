Eagle MultiView 1.5.1

使用方法
1. 先启动 Eagle 4.0，并打开要浏览的资料库。
2. 从“应用程序”启动 Eagle MultiView。
3. 点击右上角“＋窗口”或按 Command-N 新建独立窗口。
4. 每个窗口可以分别停留在不同文件夹、搜索或筛选结果。

目录浏览
- 左侧文件夹按真实父子层级嵌套显示；每个新窗口默认全部收起。
- “资料库根目录”显示一级文件夹和未分类的根目录文件；“全部素材”显示全库素材。
- 打开文件夹后只显示直属子文件夹和直属文件，不递归混入后代图片。
- 双击文件夹卡片，或单击选中后按 Return 进入。
- 可使用返回、前进、上一级、路径导航，以及 Option-左/右/上方向键。
- / 与 Eagle 原版一致：根据当前层级在“全部展开”和“全部收起”间切换；展开时会定位并高亮当前文件夹。
- 方向键在卡片间移动；Space/Return 预览素材，Command-O 用默认应用打开。
- TXT 可直接预览和编辑，Command-S 保存；图片、视频、音频、PDF 等可直接预览，PSD 等使用 Eagle 缩略图。
- 右键素材可打开原文件、在 Finder 中显示、复制路径/名称或移入废纸篓。
- Space 打开和关闭预览；预览中用左右方向键切换后，关闭时会选中最后看到的素材。
- Command-Shift-L / Command-Shift-I 可收起文件夹栏 / 检查器。

导入与整理
- 点击工具栏“导入”、按 Command-Shift-O，或把文件拖到中央素材区即可导入。
- 在 Finder 复制一个或多个文件后，回到素材区按 Command-V 可直接导入；剪贴板中的图片也可导入。
- 在普通文件夹中导入时进入当前文件夹；在根目录、全部素材、智能文件夹等视图中导入为未分类素材。
- 素材可在当前普通文件夹内置顶；置顶顺序保存在 MultiView 自己的状态中，并实时同步到所有 MultiView 窗口，不改写 Eagle 资料库元数据。
- 标签使用可添加、粘贴和逐个移除的标签块；支持逗号、中文逗号或 Return 快速添加。

数据安全
- MultiView 不会直接修改 Eagle 的 metadata.json。
- 素材属性、归类和导入都交给当前运行的 Eagle Web API 保存。
- TXT 保存会先检查文件是否被其他窗口或应用改动，并在 MultiView 的应用数据目录创建备份后再原子替换；不改写 metadata.json。
- 同一素材的写入会排队；保存前会检查另一个窗口是否改了同一字段。
- 未保存修改在切换素材、切换文件夹或关闭窗口前会被拦截确认。
- 红色关闭按钮、Command-W 和 Command-Q 都会显示明确的未保存确认。
- 标签和文件夹增删会在最新值上合并，避免旧数组覆盖新值。
- 一个 MultiView 窗口保存后，其他窗口会立即收到更新。
- Eagle 原生窗口中的修改会自动检测并刷新，通常约 1 秒触发。
- Eagle 断开时编辑控件会禁用，不会离线积压后擅自回写。
- 删除仅移入 Eagle 废纸篓，不提供永久删除。

当前支持
- 文件夹、智能文件夹、最近使用、随机模式、回收站、快速访问和未分类浏览
- 名称/标签/格式/评分筛选与搜索
- 图片、GIF、SVG、视频、音频、PDF、TXT 预览；TXT 可编辑
- 修改名称、标签、评分、备注、来源网址
- 拖到文件夹归类、批量添加标签、文件选择/拖拽/复制粘贴导入、当前文件夹置顶、移入废纸篓

1.5.1 功能更新
- 右键菜单改为 Eagle 风格浮层，补齐文件夹、标签、评分、置顶、回收站和恢复操作。
- 工具栏筛选改为弹出面板，统一管理关键词、格式、评分和标签。
- 检查器按预览、基本信息、备注与来源、文件信息分区，并显示格式标志。
- 图片缩略图不再显示 PNG/JPG 等格式角标；PSD、TXT、PDF 和音视频继续显示。
- 图片预览支持适应窗口、实际大小、缩放、拖动和当前序号。
- 修复高纵向图片在适应窗口模式下被放大裁切的问题。
- 标签颜色保存在 MultiView 自己的应用数据目录，不写入 Eagle 资料库。

1.5.1 界面更新：背景、侧栏和内容区配色对齐 Eagle 原版的中性深灰风格，并复用 Eagle 原始界面图标。

1.5.0 界面更新
- 使用原创鹰系萌娘与双窗口形象替换旧的 M 图标；Finder、Dock 和标题栏统一为新图标。
- 标题栏为 macOS 窗口按钮预留更宽的安全间距。
- 重做标题栏、工具栏、侧栏、素材网格、检查器、状态栏和预览界面的视觉层级。
- 左侧文件夹与特殊入口改为统一的矢量图标，选中、悬停和定位状态更清晰。

注意
- 需要 Eagle 4.0 Build 21 或更新版本。
- Eagle 切换资料库时，所有 MultiView 窗口会统一跟随。
- 切换资料库会清空旧库缓存，排队写入也会再次核对资料库，避免跨库误写。
- 插件、资料库合并/修复、永久删除等管理功能继续在 Eagle 中使用。
- 此版本为本机临时签名版本；若 macOS 首次拦截，请在 Finder 中右键应用并选择“打开”。
